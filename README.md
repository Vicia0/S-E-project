To run the project
1. Make sure Django is installed (pip install django)
2. Optional()
   Install virtual environment (virtualenv venv)
   Run (venv\Scripts\activate)
   Run django inside (pip install django)
3. \webproject
4. Run command
   py manage.py runserver
